🚀 Démarrage rapide (3 étapes)

1
Installation : Double-cliquez sur 1_INSTALLER.bat (une seule fois)
Installe Python et toutes les dépendances nécessaires
2
Configuration : Double-cliquez sur 3_CONFIGURER-CLE-API.bat
Configure votre clé OpenAI de manière sécurisée
3
Utilisation : Double-cliquez sur 2_LANCER.bat
Lance l'application dans votre navigateur

🔑 Obtenir une clé API OpenAI

Étapes détaillées :

  1. Aller sur le site : platform.openai.com/api-keys
  2. Créer un compte : Utilisez votre email ou connectez-vous avec Google/GitHub
  3. Vérifier votre compte : OpenAI peut demander une vérification par SMS
  4. Ajouter un moyen de paiement : Nécessaire pour utiliser l'API (même avec crédits gratuits)
  5. Créer une clé : Cliquez sur "Create new secret key"
  6. Nommer votre clé : Par exemple "Transcripteur Audio"
  7. Copier la clé : Elle commence par sk-proj- ou sk-
  8. Configurer : Utilisez 3_CONFIGURER-CLE-API.bat pour la sauvegarder

⚠️ Important :

  • Gardez votre clé API secrète
  • Ne la partagez jamais
  • OpenAI facture à l'utilisation (environ 0,006$ par minute d'audio)
  • Vous pouvez définir des limites de dépenses dans votre compte OpenAI

📊 Fonctionnalités avancées

🔄 Traitement parallèle

Le transcripteur peut traiter plusieurs fichiers simultanément pour gagner du temps. Vous pouvez ajuster le nombre de fichiers traités en parallèle dans les paramètres (1-5 fichiers).

💾 Export automatique

Vos transcriptions sont automatiquement sauvegardées dans le dossier exports/export_AAAAMMJJ_HHMMSS/ avec horodatage. Chaque export contient :

  • 📄 Transcriptions (.txt)
  • 📝 Résumés (si activés)
  • 📊 Métadonnées du traitement

🎯 Résumés intelligents

📄 Résumé bref

2-3 phrases essentielles qui capturent l'idée principale du contenu audio.

🔹 Points clés

Liste structurée et hiérarchique des informations importantes avec puces.

📋 Analyse complète

Bilan détaillé avec résumé général, points principaux et conclusion.

🎨 Interface moderne

  • Design responsive et moderne
  • Édition des transcriptions en temps réel
  • Système d'onglets pour organiser les résultats
  • Boutons de téléchargement individuels et groupés
  • Cache intelligent pour éviter les re-calculs

⚠️ Résolution de problèmes

🔴 "Python non détecté"

Solution :

  1. Installez Python depuis python.org
  2. ⚠️ Important : Cochez "Add Python to PATH" lors de l'installation
  3. Redémarrez votre ordinateur
  4. Relancez 1_INSTALLER.bat

🔴 "Clé API invalide"

Solutions :

  • Vérifiez votre clé sur platform.openai.com
  • Assurez-vous que votre compte OpenAI est actif
  • Vérifiez que vous avez un moyen de paiement configuré
  • Relancez 3_CONFIGURER-CLE-API.bat

🔴 "Fichier trop volumineux"

Solutions :

  • Compressez votre audio (utilisez Audacity ou VLC)
  • Convertissez en MP3 (format plus compact)
  • Coupez votre fichier en plusieurs parties
  • Réduisez la qualité audio (128 kbps suffisent)

🔴 "Application lente"

Optimisations :

  • Réduisez le nombre de fichiers simultanés (paramètres)
  • Fermez les autres applications
  • Vérifiez votre connexion internet
  • Redémarrez l'application

🔴 "Erreur de transcription"

Causes communes :

  • Audio de mauvaise qualité
  • Format non supporté
  • Fichier corrompu
  • Connexion internet instable

🔴 "Page blanche"

Solutions :

  • Attendez quelques secondes (chargement initial)
  • Actualisez la page (F5)
  • Vérifiez que 2_LANCER.bat est toujours ouvert
  • Redémarrez l'application

💡 Conseils pro

🎙️ Qualité audio

  • Enregistrez dans un environnement calme
  • Utilisez un micro de qualité si possible
  • Évitez les bruits de fond
  • Parlez clairement et distinctement

⚡ Performance

  • Fermez les autres applications gourmandes
  • Utilisez une connexion internet stable
  • Traitez les gros fichiers un par un
  • Surveillez l'utilisation de la RAM

💾 Sauvegarde

  • Activez l'export automatique
  • Sauvegardez régulièrement vos exports
  • Nommez vos fichiers de manière claire
  • Gardez une copie de vos audios originaux

🔄 Workflow

  • Organisez vos fichiers par projet
  • Utilisez les résumés pour un aperçu rapide
  • Éditez les transcriptions si nécessaire
  • Exportez en ZIP pour partager facilement

💰 Coûts

  • ~0,006$ par minute d'audio transcrite
  • Définissez des limites dans votre compte OpenAI
  • Surveillez votre consommation
  • Optimisez la durée de vos enregistrements

🔧 Maintenance

  • Nettoyez le dossier temp/ régulièrement
  • Mettez à jour vos dépendances si nécessaire
  • Redémarrez l'app en cas de problème
  • Sauvegardez vos paramètres importants

🆘 Support technique

Si vous rencontrez des problèmes non résolus par ce guide :

  2. 🔄 Redémarrez complètement l'application
  3. 🔧 Vérifiez que tous les fichiers .bat sont dans le même dossier

  5. 💻 Redémarrez votre ordinateur en dernier recours
